# Audi-F1-Qualifying-Exercise

### Overview

This application reads the Australian Grand Prix Qualifying lap times dataset and allows a user to enter a driver code (e.g. VER, NOR, HUL).

The application displays:

- Best valid lap time in Q1
- Best valid lap time in Q2
- Best valid lap time in Q3
- Final qualifying classification position

The solution is implemented as a Python console application.

#### Requirements:
- Python 3.11+
- pandas

Install dependencies:
pip install -r requirements.txt


#### Running the Application

From the project root:

python src/main.py
